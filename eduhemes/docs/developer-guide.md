## 🎯 Kirish

EduHemis - bu universitetlar uchun mo'ljallangan **to'liq funksional** axborot tizimi. Loyiha frontend qismi **HTML, CSS, JavaScript** va **Tailwind CSS** da yozilgan. Ma'lumotlar **LocalStorage** da saqlanadi.

### Texnologik stack

| Texnologiya | Versiya | Maqsad |
|-------------|---------|--------|
| HTML5 | - | Sahifa tuzilishi |
| CSS3 | - | Dizayn va stillar |
| Tailwind CSS | 3.x | Tezkor stillash |
| JavaScript | ES6+ | Interaktivlik |
| Font Awesome | 6.x | Ikonkalar |
| Chart.js | 4.x | Grafiklar |
| FullCalendar | 5.x | Kalendar |
